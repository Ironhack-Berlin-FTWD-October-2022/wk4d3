Async / Await

Why use Async/Await Functions?

  • The async and await keywords provide a better syntax for working with Promises and allow you to write asynchronous code as if it was synchronous. These two keywords allow us to suspend the execution of the code while we await for asynchronous code to finish.

IMPORTANT!!!!!

Async functions return a promise (not a value!)

The Synchronous approach!

function doSomething(){

const data = longLongLongRunningFunction()//Program Stuck waiting for this line!

}

The Asynchronous approach!

async function doSomething(){

const data = await longLongLongRunningFunction()//Program Stuck waiting for this line!

}
